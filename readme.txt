BMP file viewer

Показывает основную информацию о файлах формата bmp, отрисовка картинки

Требования
Python версии не ниже 3.5
PyQt 5

Состав:
файл для запуска - bmp.py
основные структуры и парсинг - bmp_core.py
графический интерфейс - gui.py
отрисовка картинки - rendering.py
тесты - tests.py

Справка по запуску
вызов встроенной справки: bmp.py -h
параметры:
*--gui - запуск графического интерфейса
*-f <file> указание пути к файлу
пример запуска: bmp.py -f file.bmp --gui

Справка по графическому интерфейсу:
*Функции "Open" и "Close" для выбора файла и закрытия программы соответственно
*"Show palette" отображает палитру файла (если присутствует)
*"Show info" скрывает или показывает информацию о файле
